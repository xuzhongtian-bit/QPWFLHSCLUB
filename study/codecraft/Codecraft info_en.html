Codecraft description

proprieter：Rex

deputy director：Tim

adviser：Eric

description：

  • "Codecraft is a student-led, project-driven tech club dedicated to enhancing members’ programming skills, project collaboration abilities, and technical literacy through real-world development projects. Weekly group meetings are scheduled by the school and revolve around the TOWER project list, covering task progress reviews, requirement discussions, priority adjustments, and Q&A sessions. Daily development work is carried out individually during members’ free time. Members collaborate on code through Git version control and discuss technical issues and provide support via a WeChat group."

The club adopts a dual-driven model of "project-based development + weekly meeting collaboration":

  • The weekly collective meeting is uniformly scheduled by the school and is centered around the TOWER project list, covering task progress review, discussion on additional requirements, priority adjustment and problem-solving Q&A, etc.
  • Daily development is completed by individuals in their spare time. Members participate in code collaboration through Git version control and discuss technical support and issues in the WeChat group.

Joining this club means that you will:

  • Participate directly in the development of a real project and build a website from scratch (from 0 to 1)
  • Participate in the development process and use tools (such as Git, TOWER, Node, etc.) of the team
  • Developing self-driven learning habits and rigorous engineering thinking
  • Growth into a true developer through continuous engagement in code conflicts and collaboration
